# Enterprise Mainframe DB2 Knowledge Assistant

An AI-powered enterprise knowledge platform designed for IBM DB2 for z/OS, COBOL, JCL, CICS, and Mainframe Operations.

Built using Retrieval-Augmented Generation (RAG), Hybrid Search, Vector Databases, and Large Language Models (LLMs), the platform provides intelligent, context-aware assistance for DBAs, Developers, Production Support Teams, and Mainframe Engineers.

The system enables faster troubleshooting, root cause analysis, operational knowledge retrieval, and simplified access to enterprise mainframe documentation by combining domain-specific knowledge with AI-driven semantic search.
